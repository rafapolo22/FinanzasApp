# FinanzasApp

## Especificaciones del Proyecto
- **Nombre:** FinanzasApp
- **Stack:** Python 3 + MySQL con `mysql-connector-python`
- **Idioma del código:** Español
- **Manejo de errores:** Uso obligatorio de `try/except` en todas las funciones para asegurar la robustez del sistema.

## Estructura de Directorios
- `database/`: Contiene la lógica de conexión a la base de datos y scripts SQL.
- `modules/`: Contiene la lógica de negocio y módulos funcionales.
- `main.py`: Punto de entrada de la aplicación con el menú principal.

## Convenciones
- Los nombres de variables, funciones y comentarios deben estar en español.
- Seguir las mejores prácticas de PEP 8 para el código Python.
